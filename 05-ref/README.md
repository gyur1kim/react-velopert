## 5.0 ref란?

> HTML에서 id를 사용하여 DOM에 이름을 다는 것처럼 **리액트 프로젝트 내부**에서 **DOM**에 이름을 다는 방법 **ref 개념**
>
- ref는 전역적으로 작동하지 않고 컴포넌트 내부에서만 작동함 → 중복 문제 발생X
